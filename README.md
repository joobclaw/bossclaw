## 💡 关于打赏授权（为什么采用授权制？）

BossClaw 是一款极大提升求职效率的利器。为了保障软件的长远运行和所有用户的利益，我们采取了\*\*“打赏获取授权码”\*\*的机制。

**为什么要这样做？核心是为了“防止滥用”。**

1. **规避风控与封杀：** 如果工具完全免费且无门槛开放，极易被黑产、猎头或恶意营销人员大规模滥用。这不仅会严重破坏招聘平台的生态，更会导致 BossClaw 很快被官方封杀，让真正需要的求职者无工具可用。
2. **过滤恶意用户：** 通过微薄的“打赏授权”门槛，我们能够有效阻挡恶意滥用者，确保服务器资源和工具真正服务于像您一样有迫切求职需求的普通人。
3. **持续迭代维护：** 招聘平台的规则在不断变化，软件的防封策略和功能更新需要投入大量的时间与精力，您的打赏是我们持续为您保驾护航的动力。

## 🚀 使用帮助 (快速上手)

只需简单四步，即可开启自动求职之旅：

1. **获取软件：** 下载 BossClaw 最新版本压缩包，解压到电脑本地文件夹，双击运行主程序。
2. **打赏授权：** 扫描软件内的打赏二维码（或联系客服），完成打赏后，将获取到的“授权码”填入软件并激活。
3. **配置条件：** 在软件界面设置您的求职期望（例如：期望岗位、最低薪资、必须包含的关键词、需要屏蔽的关键词等）。
4. **一键启动：** 登录您的求职账号，点击“开始运行”。接下来，泡杯咖啡，等着 HR 主动找您要简历吧！
